# SWARM

A DirectX 12 renderer and game engine personal project written in C++20.

## Features

- DirectX 12 rendering pipeline with double buffering
- Entity Component System via EnTT
- SDL3 windowing and input handling
- HLSL shaders compiled at runtime
- DirectXTK12 math and utilities

## Requirements

- Windows 10/11
- Visual Studio 2022 (or any MSVC-compatible toolchain)
- CMake 3.20+
